Updating to v.6.3.1

RapsterUK 27 Oct 2024

This update for RetroBat includes some key changes related to 'save' locations that you need to be aware of.

These actions should be done AFTER the upgrade and BEFORE you try to launch any games.

From the RetroBat change log:

Move RPCS3 dev_hdd0 folder to saves\ps3\rpcs3\hdd_0 (you need to cut/paste when upgrading, if you want to keep the current folder in emulator directory, delete saves\ps3\rpcs3\hdd_0 folder)

Move PPSSPP memstick folder to saves\psp (you need to cut/paste when upgrading)

Move CEMU mlc01 folder to saves\wiiu\cemu\mlc01 (you need to cut/paste when upgrading, if you want to keep the current folder in emulator directory, delete saves\wiiu\cemu\mlc01 folder)

Move XENIA and XENIA-CANARY content path to saves\xbox360\xenia (you need to cut/paste when upgrading, if you want to keep the current folder in emulator directory, delete saves\xbox360\xenia folder)

Move Ryujinx portable path to saves\switch\ryujinx (you need to cut/paste when upgrading, if you want to keep the current folder in emulator directory, delete saves\switch\ryujinx folder)

Move Vita3K content path to saves\psvita\vita3k as default(you can still use a custom path)

Recommended Steps:
PS3:
Move \RetroBat\emulators\rpcs3\dev_hdd0 to \RetroBat\saves\ps3\rpcs3\dev_hdd0

PSP:
Move \RetroBat\emulators\ppsspp\memstick to \RetroBat\saves\psp\memstick

WiiU:
Move \RetroBat\emulators\cemu\mlc01 to \RetroBat\saves\wiiu\cemu\mlc01 Xbox360: (Only move folders with names similar to "58410901" as these are where the save games are stored. For example, folders such as 'cache' 'config' or 'patches' are part of the emulator and show be left where they are)
Move \RetroBat\emulators\xenia\ to \RetroBat\saves\xbox360\xenia\ Note: Xenia and Xenia-Canary share they save saves folder.

Switch:
Move \RetroBat\emulators\ryujinx\portable to \RetroBat\saves\switch\ryujinx\portable

PSVita: (Optional but recommended)
1. Close RetroBat and browse to \RetroBat\emulators\vita3k\ and run Vita3K.exe
2. Click in the Vita3K window and go to 'Configuration' then 'Settings'
3. Click on the 'Emulator' tab then click 'Change Emulator Path'
4. Browse to \RetroBat\saves\psvita\vita3k and click 'Select Folder
5. Now click on 'Save' then 'Close'
6. You can now close the Vita3K emulator
7. Now move (cut/paste) the following folders from \RetroBat\emulators\vita3k\ to \RetroBat\saves\psvita\vita3k\
gro0
grw0
host0
imc0
os0
pd0
sa0
sd0
tm0
ud0
uma0
ur0
ux0
vd0
vs0
xmc0
8. All done!
