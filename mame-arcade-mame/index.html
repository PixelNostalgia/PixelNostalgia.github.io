MAME Arcade (mame)

RGS 25 Oct 2024

for Batocera and Retrobat
MAME 0.265

MAME (originally an acronym of Multiple Arcade Machine Emulator) is a free and open-source emulator designed to recreate the hardware of arcade game systems in software on modern personal computers and other platforms. Its intention is to preserve gaming history by preventing vintage games from being lost or forgotten. It does this by emulating the inner workings of the emulated arcade machines; the ability to actually play the games is considered "a nice side effect". 

Joystiq has listed MAME as an application that every Windows and Mac gamer should have.
The first public MAME release was by Nicola Salmoria on February 5, 1997. It now supports over 7,000 unique games and 10,000 actual ROM image sets, though not all of the games are playable. MESS, an emulator for many video game consoles and computer systems, based on the MAME core, was integrated into MAME in 2015.

Q&A:
1.Q What is Mame 0.265?
1.A This is the latest Mame Romset (Arcade) version for Batocera v40

2.Q What is a non-merged Romset?
2.A Non-merged (a.k.a. reference set): This is the same as full non-merged, however “common” BIOS files shared by a large number of ROMs are separated out to save space. Use this only if you know which BIOS files you'll need for each ROM.

3.Q What version of the MAME emulator does Batocera have?
3.A Batocera v40 uses Mame 0.265 Emulator, Also Batocera has different core's of mame eg: Mame 078Plus And Mame 0.139. Its important to match the Emulator version of MAME with the Romset.

4.Q Does the Romset 0.265 work with Mame Emulator 0.259 Batocera 39
4.A Yes! For 99% of the Roms, there are some new and updated Roms that does not work with backward compatibility but they didn’t work in the previous Romset so you won't miss out anyway :).

5.Q Why not use a merged Romset?
5.A In a merged set Clones are merged into the parent ROM ZIP file, meaning that more than one game is stored per file. This relies on the frontend to select which game is to be played. Batocera does not have this functionality, thus it is not recommended to use this type of set.

6.Q What about RetroBat?
6.A RetroBat 6.3.1 has the mame 0.267 emulator. Mame romset 0.265 will run 99%. We choose to be complatible with Batocera because their supported romset is a older version compared to RetroBat.

7.Q Why are there so many double roms from fbneo compared to mame?
7.A fbneo is a fork of mame, aimed to be more features rich for end users such as RetroAchievements. Fbneo is unable to play disc based games like Killer Instinct. The Goal of MAME is preservation. Mame will support more games and disc based games.

Requirements: Batocera v40 with the emulator set to MAME or Libretro:MAME (not on auto!) RetroBat 6.3.1

How to install:
If you are on Batocera v39 don't replace mame 0.259
If you are on Batocera v40 remove roms\mame and replace with this 0.265
If you are on RetroBat remove roms\mame and replace with this 0.265

How to install Lite pack:
If you are on Batocera v40 remove roms\mame and replace with this (0.265) If you are on RetroBat 6.x remove roms\mame and replace with this (0.265) This pack is NOT for Batocera v39 but you can still use it if you wish, 99% of games should work fine.

Requirements for realistic bezels: Have a MAME Pack installed (This can either be MAME 0.259, 0.265 or the Lite Pack).

How to install realistic bezels:
1. (Batocera): Drag and drop the folder "ArcadeRealistic" from the zip file into \\batocera\share\decorations
(RetroBat): Drag and drop the folder "ArcadeRealistic" from the zip file into R:\RetroBat\decorations

2. Go into the Mame Wheel, press select on your controller > Advanced System Options > Decorations > Select ARCADEREALISTIC

3. Done, launch a game eg: Galaga and see the Arcade Realistic Bezel. Arcade games without a Realistic Bezel will fallback to the default Arcade Bezel
