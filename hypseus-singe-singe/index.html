Hypseus SINGE (singe)

g-spawn 26 Oct 2024

for Batocera and Retrobat

Hypseus SINGE, Somewhat Interactive Nostalgic Game Engine, (named after the dragon from Dragon's Lair) is the logical evolution of the Daphne emulator, which uses a Lua-based scripting system to quickly simulate LaserDisc games or create completely new games based on video files! The language is easy to learn, very powerful and fast. All the functions needed to develop your own game are available through a simple application programming interface (API).

The package includes:

91 games (~109 GB)

Go to download    

Many Thanks to g-spawn for reaching out to the comunity and putting together this pack! Thnx RapsterUK for creating the YT video. 

Hypseus is a fork of Matt Ownby's Daphne.
A program to play laserdisc arcade games on a PC, Mac or Raspberry Pi.
This version includes Singe support for Fan Made and American Laser Games.

How to Install:
1. Drag and drop the "singe" folder to the "roms" directory
2. If asked to overwrite say yes to all
